# Sysadmin

## Prerequisites

- [docker](https://docs.docker.com/engine/install/ubuntu/)
- [docker-compose](https://docs.docker.com/compose/install/)
- [nvidia docker](https://github.com/NVIDIA/nvidia-docker)

## How to start


clone repo and run:
```
docker-compose up [services]
```

available services:

- ***master***
- ***camera0***
- ***camera1***
- ***camera2***
- ***camera3***
- ***server1***
- ***server2***
